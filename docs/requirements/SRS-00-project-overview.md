# Virtual Human Chat - プロジェクト統合SRS

## 背景/目的
- Gemini Live APIを活用したリアルタイム音声対話システムの構築
- React + BFF + Firebase/Cloud Run アーキテクチャでの高品質音声チャットボット実現
- 日本語音声に最適化された自然な対話体験の提供

## プロジェクト概要
- **技術スタック**: React + TypeScript + shadcn/ui + Express + Firebase Hosting + Cloud Run
- **リージョン**: asia-northeast1 (東京)
- **想定ユーザー**: 音声による自然な対話を求める日本語話者

## 統合受け入れ基準

### 基本機能
- ✅ ユーザーがマイクボタンで音声入力できる
- ✅ AIが日本語音声で応答を返す
- ✅ 音声遅延が1秒以内
- ✅ 接続状態が明確に表示される
- ✅ エラー発生時に適切なメッセージが表示される

### セキュリティ
- ✅ HTTPS通信のみでアクセス可能
- ✅ APIキーがフロントエンドに露出しない
- ✅ 音声データが一時保存されない
- ✅ 適切なレート制限が動作する

### 品質・パフォーマンス
- ✅ ページ読み込み時間3秒以内
- ✅ 音声録音開始までの遅延100ms以内
- ✅ 主要ブラウザ（Chrome, Safari, Firefox）で動作
- ✅ モバイル端末で基本機能が利用可能

### 運用・監視
- ✅ ヘルスチェックエンドポイントが応答する
- ✅ エラーログが適切に記録される
- ✅ デプロイが5分以内に完了する
